# Product Overview

This is a role-based Laravel application with four distinct user types:

- **Admin**: Administrative access and management
- **Issuer**: Issues credentials or documents
- **Student**: Receives and manages credentials
- **Verifier**: Verifies credentials or documents

The application appears to be a credential management system, likely for educational certificates or similar documents. Each role has its own dashboard and specific permissions enforced through middleware.

## Key Features

- Role-based access control (RBAC)
- User authentication with Laravel Breeze
- Modern SPA experience with Inertia.js and Vue 3
- Email verification support
- Profile management
